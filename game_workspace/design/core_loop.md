# Core Loop Design

## Primary Resource Transport Mechanics

### Stations
- Fixed infrastructure points in space
- Produce and consume specific resources
- Have storage capacity limits

### Cargo
- Resources transported between stations
- Moved by player-controlled ships
- Have quantity and type properties

### Core Loop
1. Identify resource imbalance between stations
2. Load cargo at surplus station
3. Transport to deficit station
4. Unload and collect payment
5. Repeat with optimized routes
