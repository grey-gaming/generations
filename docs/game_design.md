# Game Design Document

## Core Logistics Loop

### Overview
Generations is a logistics simulation game where players manage cargo transportation networks across generations.

### Core Loop
1. **Accept Contracts**: Players receive cargo delivery requests with origin, destination, and time constraints
2. **Plan Routes**: Players design efficient transportation routes using available vehicles and infrastructure
3. **Execute Deliveries**: Cargo is transported along planned routes, generating revenue upon successful delivery
4. **Expand Network**: Revenue is reinvested into new vehicles, routes, and infrastructure upgrades
5. **Legacy Progression**: Knowledge and assets pass between generations, unlocking new capabilities

## Resource Economy

### Primary Resources
- **Capital**: Currency earned from deliveries, used for purchases and upgrades
- **Cargo**: Goods to be transported (various types with different values)
- **Time**: Limited resource affecting contract availability and generation transitions

### Vehicles
- **Trucks**: Short-range, flexible routing, moderate capacity
- **Trains**: Long-range, fixed routes, high capacity
- **Ships**: Water-based routes, slow but high capacity
- **Planes**: Fast, expensive, limited cargo types

### Infrastructure
- **Depots**: Cargo storage and vehicle maintenance
- **Stations**: Loading/unloading points for trains
- **Ports**: Water-based cargo handling
- **Airports**: Air cargo facilities

## Success Metrics
- Delivery completion rate
- Network efficiency (revenue per time unit)
- Legacy score (multi-generational wealth and impact)
