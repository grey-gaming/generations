# Core Economy Resources and Flow

## Resource Types

### Primary Resources
- **Credits (CR)**: Main currency for transactions
- **Energy (EN)**: Powers vehicles and infrastructure
- **Materials (MT)**: Used for construction and repairs

### Secondary Resources
- **Influence (IF)**: Unlocks contracts and permissions
- **Research Points (RP)**: Drives technology progression

## Resource Flow

```
Contracts -> Deliveries -> Credits -> Infrastructure -> More Contracts
                ↓
            Energy + Materials (consumption)
                ↓
            Repairs/Refuel (maintenance cost)
```

## Economic Balance

### Income Sources
1. Contract completion bonuses
2. Route efficiency rewards
3. Legacy inheritance

### Expense Categories
1. Vehicle fuel/energy
2. Infrastructure maintenance
3. Loan repayments (if leveraged)

## Flow Constraints

- **Storage limits**: Depots have finite capacity
- **Flow rates**: Loading/unloading takes time
- **Degradation**: Assets require ongoing maintenance

## Progression Gates

- Early: Basic trucks, short contracts
- Mid: Trains/ships, multi-leg routes
- Late: Planes, legacy optimization
