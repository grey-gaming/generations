# Core Transport Logistics Loop

## Overview

This document defines the core transport logistics loop for the game design space.

## Loop Definition

The transport logistics loop consists of three phases:

1. **Acquisition**: Resources/materials are acquired from sources (mines, farms, factories)
2. **Transport**: Resources are moved through the logistics network (trucks, trains, ships, planes)
3. **Delivery**: Resources arrive at destinations (cities, factories, export hubs) creating economic value

## Design Principles

- **Simple to learn**: Basic loop (acquire → transport → deliver) is intuitive
- **Deep to master**: Network optimization, timing, and capacity management create strategic depth
- **Multiple solutions**: Each logistics problem can be solved via different transport modes and routes
- **Emergent complexity**: Simple units interacting create interesting macro-scale behaviors

## Core Metrics

- Throughput (units/time)
- Utilization (% capacity used)
- Distance/cost efficiency
- Reliability (on-time delivery rate)

## Future Considerations

- Multi-modal transport chains
- Dynamic supply/demand systems
- Infrastructure degradation/maintenance
- Player vs player competition modes
